# ichsm completion

`ichsm completion` generates shell completion scripts.

## Usage

```
ichsm completion [command]
```

Available shells:

- `bash`
- `fish`
- `powershell`
- `zsh`

Use the shell-specific help for installation details:

```
ichsm completion zsh --help
```

## Examples

Generate a Bash completion script:

```
ichsm completion bash
```

Generate a Zsh completion script:

```
ichsm completion zsh
```

Generate a Fish completion script:

```
ichsm completion fish
```

Generate a PowerShell completion script:

```
ichsm completion powershell
```
